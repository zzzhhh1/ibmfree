## 运行说明

请按照以下步骤执行安装脚本：

1.  **切换到 root 用户权限：**

    打开您的服务器终端，输入以下命令并按 Enter。

    ```bash
    sudo -i
    ```

    *此命令会将您切换到 root 用户，后续命令将以 root 权限执行。*

2.  **下载并执行安装脚本：**


    ```bash
    bash <(curl -L -s https://raw.githubusercontent.com/byJoey/ibmfree/refs/heads/main/install.sh)
    ```
